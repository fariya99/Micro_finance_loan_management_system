# Microfinance Loan Management System (Java GUI)

## 📌 Overview

The **Microfinance Loan Management System** is a Java-based desktop application designed to manage small-scale loans efficiently. It provides a user-friendly interface for handling customers, different types of loans, repayments, and generating reports.

The system is built using Object-Oriented Programming principles and includes a GUI for better usability, making it suitable for academic and real-world microfinance scenarios.

---

## 🚀 Features

### 👤 Customer Management

* Add and manage customer records
* Store personal and financial details

### 💰 Loan Management

* Support for multiple loan types:

  * Personal Loan
  * Business Loan
  * Education Loan
* Automatic EMI (Equated Monthly Installment) calculation
* Track loan status and installments

### 💳 Payment System

* Record loan repayments
* Store payment history using CSV format
* Track remaining balance

### 📊 Reports & Analytics

* Generate loan reports
* View summaries of customers and payments

### 🖥️ GUI Interface

* Interactive Java GUI
* Login system for secure access
* Search functionality for quick data retrieval
* Enhanced UI with improved colors and layout

---

## 🛠️ Technologies Used

* **Language:** Java
* **Concepts:** OOP (Inheritance, Polymorphism, Encapsulation)
* **GUI:** Java Swing / AWT
